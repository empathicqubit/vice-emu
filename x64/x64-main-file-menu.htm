Autostart disk/tape image...

Here you can select an image file of type: d64, d71, d80, d81, d82, g64, g41, x64, t64, and tap. Once selected, the image file is attached as drive 8, true drive emulation is turned off, and the first file on the disk/tape is 'LOAD'ed and 'RUN' automatically.

You can also use this option to automatically load and run prg and p00 files from the PC's file system without them being in a disk image.

If you need help with the Attach Image box, please click here.

Attach disk image

Click the drive you wish to 'insert' a disk into. The Attach Image box will pop up. There you can select an image file of type: d64, d71, d80, d81, d82, g64, g41, or x64. Please note that there is no checking between the image type and the type of drive you are emulating so it's possible to 'insert' a d71 or d81 into a 1541 drive and vice versa. If you are having problems with an image, be sure it's of the correct type for the drive you are using.

Detach disk image

If you want to remove or eject a disk from one or all of the emulated drives choose this option. Then click the drive you want to detach the disk image from. Clicking 'All' will of course eject all disks from the drives.

Flip list

Among other uses, the flip list is an excellent way to quickly change between mutliple sides of disks for use in multi-disk games.

To start, attach a disk like normal then click on the flip list option and then click 'Add current image'. Load the next image you want to add and do the same thing. Do this for all disk images you want to be part of the list.

To use the flip list, you can go to the flip list menu and select 'Attach next image' but likely the easiest way would be to use the shortcut keys Alt-N and Alt-B to swap the disks. The status bar will show which disk it just attached and when you reach the end of the list it will cycle back to the beginning.

If you want to remove disk images from the list, simply use the flip list to attach the image you want to remove then click the 'Remove current image' option or use the shortcut keys.

You can also load and save the current flip list by using the options on the menu.

Attach tape image...

Use this to insert a tape image (.tap) into the emulated C2N datasette. It brings up the Attach image box described here.

The image attach box

 

  1. This is the file selector. It works just like the standard Windows GUI file browser. VICE uses this selector GUI for several different things so the title bar may change but the functionality is the same. You can click on folders to change to a different one, click the dropdown to see the current tree view, and click the up arrow to go to the higher level folder above the current one.
    The dropdown for "Files of type:" lets you filter the files shown by file type.
    Once you have selected the file you wish to load, just click Attach to load it into the emulator.
  2. Here is the image contents pane. Once you click on a disk or tape image in the file selector, this pane will show the directory of the image selected. It looks virtually identical to the directory display ($) on a Commodore. If you wish to have VICE automatically load a PRG file from the selected disk, just double click on the file in this pane and the emulator will LOAD it.
  3. If you need to create a blank disk image you can do so here. First you must type a filename for the new image in the file selection pane where it says "File Name:". Then you type in the directory header name where it says Name and the desired disk id where it says ID. Keep in mind that the case you use here will echo the same way in the header. Normally you'll want to use all uppercase letters so they show up like most disks do in the emulator. Finally, click the dropdown to select which type of disk image you want to create then click "Create Image".

    If you are attaching a tape image, you will see "New Tap Image" instead of the items in #3 above. Simply type in a file name then click Create Image.
